# 价格之和信念验证报告
**日期**: 2026-03-02
**数据源**: 全市场parquet历史数据 (2022-12 ~ 2025-08)
**覆盖范围**: 12个品种 (SN/PB/AD/SF/UR/PF/AP/CJ/CS/JD/LH/LG), 3个交易所 (SHFE/CZCE/DCE)
**分析量**: 328个交易日, 791个配对 (近端328/中端327/远端136)

---

## B001 方向性跟随

**原始信念**: 价格之和跟随期货同方向运动（宽跨式净Delta为正，因期货价格离Call行权价更近）
**原始置信度**: 0.75
**验证结果**: +393 支持 / -278 反驳 / ~120 中性 → 支持率 58.6%
**新置信度**: 0.75 (不变)

**分析**:
- 期货涨时价格和确实倾向于涨,但**并非压倒性**。58.6%的支持率刚刚过半
- 反驳的278次中,很多是**期货小涨但价格和下跌**的情况——说明theta/vega的影响可以压过小幅方向性
- 在期货**大幅运动**(>1%)时,方向性跟随更可靠;小幅震荡时方向不确定

**建议修改**: 原statement过于绝对。建议改为:
> "价格之和在期货大幅运动(>1%)时倾向于跟随同方向运动,但在小幅震荡(<0.5%)时方向不确定,Theta/Vega可能主导"

---

## B002 下跌不对称性

**原始信念**: 价格之和对期货下跌的敏感度约是上涨的2倍
**原始置信度**: 0.55
**验证结果**: +278 支持 / -346 反驳 / ~73 中性 → 支持率 44.6%
**新置信度**: 0.55 (不变)

**分析**:
- **跨品种验证不支持"2倍"这个比例**。在12个品种中,下行/上行敏感度比值>1.5的只占40%
- 很多品种的比值在0.8~1.2之间,即**上下行基本对称**
- 这个信念最初来自ag2604单一配对的单一夜盘数据,可能是特定行情下的偶发现象
- **ag的不对称性可能是因为期货大涨3.9%,导致Call向ATM靠拢而Put更加深虚值**——这是特定行情的结构效应,不是普遍规律

**建议修改**:
> "不对称性**不是**普遍规律。它取决于当前期货价格相对行权价的位置和行情幅度。只有在期货大幅单边运动后,因Call/Put的虚值程度变化不均匀才会出现。ag单夜盘观察到的2倍比值是极端案例"

## B003 虚值杠杆效应

**原始信念**: 越深度虚值的配对,百分比波动越大
**原始置信度**: 0.70
**验证结果**: +439 支持 / -160 反驳 → 支持率 73.3%
**新置信度**: 0.71 ▲

**分析**:
- **强有力的跨品种验证**。近3/4的情况下,更深虚值的配对确实有更大的百分比波动
- 这是数学必然:分母(总权金)更小,同样的绝对变化会产生更大的百分比
- 反驳的27%可能来自**流动性差异**——极深虚值期权有时tick大、数据稀疏

**建议**: 保持原表述不变,置信度合理上升

## B004 Delta结构递减

**原始信念**: 近端Delta > 中端Delta > 远端Delta
**原始置信度**: 0.60
**验证结果**: +477 支持 / -122 反驳 → 支持率 79.6%
**新置信度**: 0.62 ▲

**分析**:
- **强验证**。近80%的情况符合Delta递减结构
- 这也是理论必然:越深虚值的期权Delta越小,宽跨式净Delta越小
- 但绝对值(0.076/0.156/0.533)只适用于ag2604的特定时刻,不应当作通用参数

**建议**: statement中去掉具体数字,改为:
> "宽跨式净Delta与虚值程度负相关:近端净Delta最大(通常>0.3),中端次之(0.05~0.2),远端最小(<0.05)。具体值随期货价格和DTE变化"

## B005 三因素驱动

**原始信念**: 价格之和有三个驱动:①方向性(Delta) ②波动率(Vega) ③时间衰减(Theta)
**原始置信度**: 0.80
**验证结果**: +544 支持 / -0 反驳 → 支持率 100%
**新置信度**: 0.82 ▲

**分析**:
- 所有session的回归残差都是负值(avg_residual < 0),说明**去掉方向性后,价格和有持续的向下漂移**——这就是Theta衰减的证据
- 100%支持率,但需注意这个验证方法本身偏向支持(负残差≈theta存在)

**建议**: 保持不变。这是期权定价理论的必然结果,高置信度合理

## B006 弹簧效应

**原始信念**: 深虚值价格和像弹簧——被Theta压缩到低位后,大行情使其弹起
**原始置信度**: 0.55
**验证结果**: +27 支持 / -6 反驳 → 支持率 81.8%
**新置信度**: 0.58 ▲

**分析**:
- 样本量较小(33次,因为需要同时有近端和远端配对且期货>1%变化)
- 但在有数据的情况下,**82%的时候远端百分比涨幅确实超过近端**
- 这与B003(虚值杠杆)一致——弹簧效应本质上就是杠杆效应的交易应用

**建议**: 保持不变,但注明"弹簧效应是虚值杠杆(B003)的交易应用,非独立现象"

## B007 Put钉死现象

**原始信念**: 深虚值Put在期货持续上涨时价格几乎不变
**原始置信度**: 0.70
**验证结果**: +544 支持 / -0 反驳 → 支持率 100%
**新置信度**: 0.73 ▲

**分析**:
- **非常强的验证**。在791个配对中,544个检测到了Put钉死窗口(期货涨>0.3%但Put变<2%)
- 这说明Put钉死不是ag特有现象,而是**跨品种的普遍规律**
- 机理清晰:深虚值Put的Delta接近0,期货小幅变动不影响其定价

**建议**: 置信度可以更高。保持statement不变

## B009 极深虚值Put退化 ⚠️

**原始信念**: 极深虚值Put对期货波动完全免疫,退化为纯时间价值
**原始置信度**: 0.65
**验证结果**: +20 支持 / -37 反驳 → 支持率 35.1%
**新置信度**: 0.62 ▼

**分析**:
- **被反驳!** 在远端配对中,Put的百分比波幅往往超过预期
- 原始观察基于ag P15000(>35% OTM),极端情况。一般的"远端"(15-50% OTM)Put并非完全免疫
- "完全免疫"的表述过于极端。Put在深虚值时确实敏感度低,但不是零

**建议修改**:
> "深虚值Put(15-50%OTM)对期货波动的敏感度显著低于浅虚值,但**并非完全免疫**。只有在极端OTM(>35%)且权金已极低(<5点)时才接近完全退化。一般远端Put仍会有可观的百分比波动"

## B010 Put钉死入场信号 ⚠️⚠️

**原始信念**: 检测到Put钉死后做多价格和(买入宽跨)是好的入场窗口
**原始置信度**: 0.50
**验证结果**: +95 支持 / -388 反驳 → 支持率 19.7%
**新置信度**: 0.45 ▼

**分析**:
- **严重被反驳!** 只有20%的情况下,Pin信号后价格和是上涨的
- **80%的时候,Pin信号后价格和反而下跌**——这是因为Pin信号经常出现在趋势早期/中期,而session结束时趋势回调导致价格和下跌
- 问题在于信号定义:检测到Put钉死只说明过去10分钟期货在涨,不预示未来
- 原始信念混淆了"已发生的现象"和"未来的预测"

**建议修改**:
> "Put钉死是一个**已发生现象的描述**(过去N分钟Put不动),而非可靠的入场信号。钉死发生后,期货可能继续涨(有利),也可能回调(不利)。信号后价格和上涨的概率仅约20%,**不应单独作为交易依据**。需要结合其他指标(如趋势强度、成交量)过滤"

## B011 Put钉死→暴涨两阶段

**原始信念**: Put钉死后期货回调会触发Put暴涨,此时价格和两头赚
**原始置信度**: 0.70
**验证结果**: +192 支持 / -0 反驳 → 支持率 100%
**新置信度**: 0.73 ▲

**分析**:
- **强验证**。所有检测到Put暴涨(5分钟涨>10%)且期货同时下跌的事件中,价格和都是上涨的
- 这证实了两阶段机制:Put被压制→释放弹性→期货回调时Put大涨>Call小跌=净赚
- 232个中性说明很多session没有出现Put暴涨(这是正常的,暴涨是罕见事件)

**建议**: 保持不变。这是一个有价值的微观结构洞察

## B012 暴涨后高位平台

**原始信念**: Put暴涨后不会完全回落,会稳定在更高平台
**原始置信度**: 0.55
**验证结果**: +181 支持 / -184 反驳 → 支持率 49.6%
**新置信度**: 0.55 (不变)

**分析**:
- **几乎正好50/50**,说明"高位平台"不是普遍现象
- 有时Put暴涨后确实不回原位(波动率被重新定价),有时会回落(虚假突破)
- 可能取决于暴涨的**原因**:如果是波动率驱动(VIX上升),平台更持久;如果是流动性冲击(卖盘瞬间被吃光),可能很快回落

**建议修改**:
> "Put暴涨后是否形成高位平台取决于暴涨的驱动因素。约50%的情况下会形成高位平台(波动率重定价),50%会回落(流动性冲击)。**不能假设暴涨后的高价会持续**"

## B013 DTE≤14天theta收割

**原始信念**: 期货不动时卖出宽跨可以赚theta
**原始置信度**: 0.30
**验证结果**: +209 支持 / -58 反驳 → 支持率 78.3%
**新置信度**: 0.34 ▲

**分析**:
- **意外的强支持!** 当期货变化<0.3%时,78%的session价格和确实下降了
- 这直接证明了theta衰减的存在——期货不动时,时间在持续侵蚀价格和
- 但之前的夜盘回测(B013原始数据)胜率低,是因为那个夜盘期货大幅单边下跌,不属于"期货不动"的情况

**建议修改**:
> "当期货波动<0.3%时(震荡市),价格之和有78%概率下降(theta占主导)。但**单边行情时theta收割不可行**——方向性/Vega冲击远超theta收益。关键前提是判断行情是否为震荡市"

### 最重要的发现

1. **B010需要重大修改**: Put钉死不是可靠的入场信号(仅20%胜率),原始假设被数据严重反驳
2. **B002不是普遍规律**: 下行/上行不对称性取决于特定行情结构,不应作为通用假设
3. **B009过于极端**: "完全免疫"表述不准确,应改为"敏感度显著降低"
4. **B013有转机**: 在"期货不动"的条件下,theta收割的78%胜率比预期好得多
5. **B007/B011是最有价值的发现**: Put钉死现象和两阶段信号都得到了强烈的跨品种验证
